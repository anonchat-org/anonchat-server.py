anonchat - a microprotocol for communication built using TCP sockets.

A server acts as a message distributor between the connected clients, while a client can send and recieve messages.

A typical anonchat message is just a JSON encoded object which contains message text and (optionally) its author. The socket has only one event: an incoming message. The protocol is simple as hell. It's very easy to configure, extend and hack.

Example message object: {"user": "calamity", "msg": "hey, how you doing today?"}
Pretty simple, isn't it?

Also, there is a client and server protocol implementations written on various languages, like Dart, Python, Rust, etc. You can see the source codes here.
